## 前言 React 介绍
&emsp;React 是 Facebook 于 2013 年推出的函数式编程，是目前前端框架中使用人数最多的前端框架，其拥有健全的文档与完善的社区。React 16.0 及以上版本，又被成为 React Fiber。

<br>

## 引用
1. 引用 .js 文件来使用 React
2. 通过脚手架工具来编程

&emsp; 这里通过官方提供的脚手架工具： create-react-app
```
    npm i create-react-app -g
    create-react-app my-app

    cd my-app
    npm start
```

> 此刻， ** ** 的 `node` 等版本为：  
> `node` : 8.11.2  
> `npm` : 6.3.0  
> `cnpm` : 6.0.0
